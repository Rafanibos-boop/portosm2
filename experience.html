Building My
Portfolio Website

A personal account of how I designed and developed my portfolio website as an RPL student — combining creativity, code, and collaboration.

🌱
Orientation

The Beginning — 01

Last month, I got a collaboration assignment in my RPL class to create a portfolio website using HTML, CSS, JavaScript, and Figma. The assignment was done individually, but we still collaborated with our classmates.

⚡
Events

The Process — 02

First, I designed the website layout in Figma. After the design was finished, I started building the website structure using HTML and improved the appearance with CSS. Then, I added some interactive features using JavaScript to make the website more attractive.

During the process, I faced several difficulties, such as coding errors and problems with arranging the website layout. However, I discussed and exchanged ideas with my classmates so the problems could be solved. We helped each other even though the assignment was done individually.

✅
Reorientation

The Takeaway — 03

In the end, I successfully completed the portfolio website and presented it in front of the class. From this experience, I learned that collaboration does not always mean working in groups, but it can also be done by helping and sharing ideas with others.

Collaboration does not always mean working in groups — it can also be done by helping and sharing ideas with others.

— Rafa Febrianyah, PPLG Student 2026
